# BBT Friday Flash Sale Runner

Run every Friday by 14:00 to prepare for 15:30 launch.

## Instructions

STEP 1 — SELECT PRODUCT
If no product specified, query Shopify for:
- High margin products (accessories preferred)
- Good stock levels (minimum 10 units)
- Never batteries (see OPERATIONS.md — never discount batteries)
- Not featured in last 2 flash sales
- Recommend top 3 options with reasoning

STEP 2 — SET UP IN SHOPIFY (once product confirmed)
- Add tag: flash-sale to the product
- Set compare-at price = current selling price
- Set sale price = selling price minus agreed discount
- Confirm GP floor is maintained (see OPERATIONS.md)
- Verify flash sale section on homepage is pulling product

STEP 3 — WRITE BROADCAST MESSAGES (3 versions)

MESSAGE 1 — Friday 15:30 (launch):
Format:
🔥 FLASH SALE — BBT [PRODUCT NAME]

[One line value proposition]

Was: R[compare-at]
NOW: R[sale price]
Save: R[saving amount]

👉 betterbattery.co.za/products/[handle]

⏰ Sale ends Sunday 17:30
WhatsApp: 063 500 1740

MESSAGE 2 — Saturday 09:00 (follow-up):
Shorter. Last chance angle.
Reference Sunday deadline.

MESSAGE 3 — Sunday 17:30 (close):
3 lines maximum.
"Ends tonight" urgency.
Link only.

STEP 4 — CONFIRM CHECKLIST
- [ ] Product tagged flash-sale in Shopify
- [ ] Sale price set correctly
- [ ] GP floor maintained
- [ ] Homepage flash sale section showing product
- [ ] All 3 broadcast messages written
- [ ] Sunday removal reminder set

Output all 3 messages ready to copy-paste into WhatsApp.
