# BBT Daily Business Pulse

Run this every weekday morning before 9AM.

## Instructions
Query Shopify live and report:

1. ORDERS (last 24 hours)
   - Total orders count
   - Total revenue
   - Average order value
   - List each order: order number, product, value, status

2. REVENUE
   - Today vs same day last week
   - MTD vs last month same period
   - Best selling product last 7 days

3. STOCK ALERTS
   - Any product with inventory below 5 units
   - Any product marked as sold out

4. UNFULFILLED ORDERS
   - List all orders not yet fulfilled
   - Flag any over 24 hours old

5. ACTION LIST
   - Top 3 things that need attention today
   - Any reorders needed based on stock levels

Format as a clean daily briefing. Be direct and specific.
Use R for currency. Flag anything urgent in capitals.
