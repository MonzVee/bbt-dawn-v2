# BBT WhatsApp Broadcast Writer

Use to write any WhatsApp broadcast message for BBT.

## BBT broadcast voice and rules
- Direct and energetic — no corporate speak
- Township-aware — speak to taxi operators and Uber drivers
- Price-forward — lead with the deal
- Urgency without desperation
- Never discount batteries
- Always include WhatsApp number
- Always include website link
- Maximum 150 words per message
- Use emojis sparingly — maximum 5 per message

## Instructions

Collect this information first:
- Purpose: flash sale / new product / announcement / seasonal
- Product (if applicable): name and price
- Offer (if applicable): discount amount or percentage
- Deadline (if applicable): when does offer end
- Primary CTA: website link or WhatsApp

Then write 3 versions:
- Version A: price-led (lead with the deal)
- Version B: benefit-led (lead with what customer gets)
- Version C: urgency-led (lead with scarcity or deadline)

Each version must include:
- Opening hook (first line is everything — make it stop the scroll)
- Core offer or message
- Clear CTA with link
- WhatsApp number: 063 500 1740
- Website: betterbattery.co.za

Present all 3 versions ready to copy-paste.
Muneer selects which version to send.
